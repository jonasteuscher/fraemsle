## Fraemsle

### Running locally

1. Clone this project

```sh
git clone [https://github.com/jonasteuscher/fraemsle.git](https://github.com/jonasteuscher/fraemsle.git)
```

2. Change into the directory and install the dependencies

```sh
cd fraemsle

npm install
```

3. Run the server

```sh
npm run dev
```
